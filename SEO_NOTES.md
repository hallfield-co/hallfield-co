# SEO Launch Notes

Production domain used for canonical metadata, robots, and sitemap:

- `https://www.hallfield.co/`

Recommended launch checks:

- Add the final domain to Google Search Console.
- Submit `https://www.hallfield.co/sitemap.xml`.
- Run the page through PageSpeed Insights.
- Compress new PNGs with ImageOptim or a similar lossless PNG optimizer before committing them.
